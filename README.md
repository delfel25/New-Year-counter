**New Year counter - it counts the time on your computer along with the date and tells you how much time it is until New Year's**


![photo of what it looks like](img/mainphoto.png)

---

* Language: Golang

* Rendering: ANSI and ASCII Art escape codes

* Scaling: Dynamic matrix grid calculation (digital)

### Requirements
* [Go](https://go.dev/doc/install) installed (version 1.20 or higher).

``` bash
# Clone the repositories
git clone [https://github.com/delfel25/New-Year-counter.git](https://github.com/delfel25/New-Year-counter.git)

# Go to the project address
cd New Year's counter

# Build the project
go create main.go

# Run the program
./main
